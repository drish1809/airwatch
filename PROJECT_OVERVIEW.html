🧩 Module Breakdown

📡 collector.py

  • 12 global cities from config YAML
  • OpenWeather Air Pollution + Weather APIs
  • 3× retry with delay on HTTP failure
  • Rate limiter between city calls
  • SQLite UNIQUE constraint — no duplicates
  • APScheduler daemon thread (every 30 min)
  • Auto CSV export after each run

🔧 processor.py

  • Deduplication on (timestamp, city)
  • City-median then global-median imputation
  • ±3σ winsorisation for key pollutants
  • Calendar features: hour, day, rush_hour
  • Ratio features: PM ratio, NO/NO₂
  • Lag features: pm2_5 & aqi (lags 1–3)
  • Rolling mean & std (window = 3)

🤖 trainer.py

  • 6 models: LR, Ridge, Lasso, RF, GB, ET
  • StandardScaler on all features
  • 80/20 train-test split
  • 3-fold cross-validation per model
  • Auto-selects best by RMSE
  • RandomizedSearchCV for top 3 models
  • Saves pkl + scaler + metadata JSON

🔬 anomaly.py

  • Isolation Forest (5% contamination)
  • Trained on PM2.5, PM10, NO₂, O₃, AQI
  • Flags 5% most anomalous readings
  • Anomaly score per record
  • Spike alerts: delta ≥ 50 AQI in one step
  • 3 severity levels: MEDIUM/HIGH/CRITICAL

📊 app/main.py

  • Dark theme with custom CSS + Google Fonts
  • Auto-refresh every 10 minutes
  • Cached data loading (TTL = 10 min)
  • 6 full-featured pages
  • Plotly: line, bar, scatter, heatmap, radar, gauge, map
  • Interactive sidebar filters

🛠️ run.py (CLI)

  • --demo generate data + train
  • --collect live API collection loop
  • --train re-train on latest data
  • --app launch Streamlit
  • --all collect + train + app
  • Graceful SIGINT/SIGTERM shutdown
⚡ Quick Start Commands
bash — Setup
# 1. Create virtual environment
$ python -m venv venv
$ source venv/bin/activate

# 2. Install dependencies
$ pip install -r requirements.txt

# 3. Demo mode (no API key needed)
$ python run.py --demo
✓ Generated 17,292 synthetic records
★ Best model: LinearRegression R²=1.00
bash — Launch Dashboard
# Launch Streamlit app
$ python run.py --app
🌐 Launching Streamlit dashboard…
Local URL: http://localhost:8501

# Or with live API data (add key to config.yaml)
$ python run.py --all
📡 Collector running every 30 min
🤖 Model training complete
🌐 Dashboard live
bash — Docker Deploy
# Build image
$ docker build -t airwatch .

# Run container
$ docker run -p 8501:8501 airwatch

# With your API key
$ docker run -p 8501:8501 \
-e OWM_KEY=your_key airwatch

🚀 Deployment Options

☁️ Streamlit Cloud FREE

  1. Push project to GitHub
  2. Visit share.streamlit.io
  3. Set entrypoint: app/main.py
  4. Add API key in Secrets (optional)
  5. Click Deploy — done in 2 min

🐋 Docker SELF-HOST

  1. Add Dockerfile to project root
  2. docker build -t airwatch .
  3. docker run -p 8501:8501 airwatch
  4. Deploy to any VPS or ECS/GCR
  5. Add env var for API key

🤗 HuggingFace Spaces FREE

  1. Create Space with Streamlit SDK
  2. Upload all project files
  3. Set app_file = app/main.py
  4. Add HF secrets for API key
  5. Persistent storage via HF Datasets

☁️ Render FREE TIER

  1. Connect GitHub repo to Render
  2. Build: pip install -r requirements.txt
  3. Start: streamlit run app/main.py
  4. Set environment variables
  5. Auto-deploy on git push
